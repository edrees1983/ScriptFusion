If you've started learning Scratch and want to make an app or website with it then we are gonna use TurboWarp for the job. First, make a project in Scratch (must support mobile if your making a mobile app). Then we go to turbowarp.org, after that we press File and press Package Project. To make it look professional check "Start automatically instead of showing big green flag". Then you can make it into a html (website) file, zip (also websites), Windows exe, MacOS apps, Linux apps and more. If the file you downloaded is HTML, then you can use a html to app tool, or if you use Android, download App Builder by Serakont (Its named App Builder but make sure it's made by Serakont),make a html app and load your html file in assets. Then press the play icon and your done! If you want to make an iOS app (you need a mac with Xcode, you don't need to pay if you're making a private app) or want to use Median.co, do the TurboWarp trick i showed you and then load the HTML in a GitHub repository (make sure its public) then go to settings, pages and set the branch to main and save it, then use median.co and make a project, if it asks you for an email you don't really need it so you can type your scratch username or whatever you want as long as it ends in @gmail.com (Don't worry they won't email you) after you're done go on and do some changes on app icon, then you can download .apk, .aab and source code. If you want to make an iOS app, download the source code and load it in xcode, then compile an ipa (it will expire after 7 days so you have to reinstall it every 7 days) and then you are done! Now after you've gained the skills of making apps with Scratch, then you can continue to making 3D in Scratch
